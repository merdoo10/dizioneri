If you're looking for great series to watch, you're in the right place. To save you the time of navigating endless catalogs trying to decide what to watch, I've created a genre-based recommendation guide.

Series Recommendations is a static information site, entirely developed with HTML and CSS, designed for those who want quick and carefully selected series recommendations sorted by genre, rather than an algorithm-driven feed. On the site, I've selected a short list of featured series for each genre, written a brief summary for each, and provided direct links to where they can be watched.

The project consists of this repository, a landing page where you choose a genre (series.html), and six dedicated genre pages (drama.html, comedy.html, film.html, fantasy.html, thriller.html, crime.html), each listing several carefully selected recommendations. (All pages share the same external style file, dizi.css.)

The main page features six genre buttons — Drama, Comedy, Science Fiction, Fantasy, Thriller, and Crime — each leading you to its own dedicated recommendation page.

Selected series with summaries: Each genre page includes a short, handwritten description for each recommended series. For example, the Drama page covers Succession, Shogun, and The Leftovers, each with a few sentences explaining what the series is about and who it's for.

"Check out" (watch) buttons: Each recommendation is paired with a button that links directly to the platform where the series can be watched (e.g., TV+, Disney+), so the recommendation becomes action-packed with a single click.

Fonts: To give each genre its own unique typographic personality while maintaining overall site consistency, I used Abel, Advent Pro, Geist Pixel, Imperial Script, Indie Flower, and Quicksand fonts from Google Fonts.

Warning!!: Each genre page ends with a short note reminding visitors that a paid subscription may be required to watch some of the suggested series.

Project Structure
